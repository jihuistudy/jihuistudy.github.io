数组常用API与分类

数组迭代器

arr.keys()返回索引值迭代器;
arr.values()返回元素值迭代器;
arr.entries()返回键值对。

1.forEach遍历数组所有元素

输出：false

输出：true

添加元素数组API

1.push() 方法将一个或多个元素添加到数组的末尾，并返回数组的新长度。原数组会改变

2.unshift() 方法将一个或多个元素添加到数组的开头，并返回新数组的长度。原数组会改变

删除元素数组API

1.pop() 方法从数组中删除最后一个元素，并返回该元素的值。此方法更改数组的长度，原数组会改变。

2.shift() 方法从数组中删除第一个元素，并返回该元素的值。此方法更改数组的长度，原数组会改变。

操作元素数组API

1.concat() 方法用于合并两个或多个数组。此方法不会更改现有数组，而是返回一个新数组。

2.slice() 方法返回一个从开始到结束（不包括结束）的索引值对应的数组元素浅拷贝到一个新数组对象，原始数组不会被修改。此方法元素会改变。

3.splice() 方法始终会返回一个数组，该数组包含从原始数组中删除的项（如果没有删除任何项，则返回一个空数组）

删除：需指定2个参数，要删除的第一项位置和要删除的项数
插入：需提供3个参数，起始位置、0（要删除的项数）和要插入的项，如要插入多个项，再传入第四，第五个参数
替换：需指定3个参数，起始位置、要删除的项数和要插入的任意数量的项

获取数组元素位置API

1.indexOf()方法从数组的开头开始向后查找,可返回某个指定的数组元素在数组中首次出现的位置（索引），不会改变原数组。

接收两个参数：要查找的数组项和查找起点位置的索引（可选，没有则默认为0），没找到返回-1

2.lastIndexOf()方法从数组的末尾开始向前查找，可返回某个指定的数组元素在数组中最后出现的位置（索引），不会改变原数组。

接收两个参数：要查找的数组项和查找起点位置的索引（可选，没有则默认从最后开始），从起点索引值向前开始搜索，没找到返回-1

这两种API也适用于字符串，用法一致

重排序数组API

1.sort排序，返回数组，且原数组改变

输出：[1,2,3,4,5,18]

注意：单独使用arr.sort()方法，没有在sort()中使用参数，将按字母顺序对数组中的元素进行排序，说得更精确点，是按照字符编码的顺序进行排序。输出会变为[1,18,2,3,4,5]

2.reverse()颠倒数组中元素的顺序，会改变原数组

转换成字符串的数组API

1.join(“连接符”) 方法用于把数组中的所有元素放入一个字符串，用自定义的”连接符”连接每个元素，如果省略连接符参数，则使用逗号作为分隔符。不改变原数组。

2.toString() 方法，把数组转换为字符串，并返回字符串，不改变原数组。

总结：哪些API会改变原数组？哪些不会？

  • 改变原数组的API
    pop()—删除数组的最后一个元素并返回删除的元素。
    push()—向数组的末尾添加一个或更多元素，并返回新的长度。
    shift()—删除数组的第一个元素并返回删除的元素。
    unshift()—向数组的开头添加一个或更多元素，并返回新的长度。
    reverse()—反转数组的元素顺序。
    sort()—对数组的元素进行排序。
    splice()—用于插入、删除或替换数组的元素。

  • 不改变原数组的API
    concat()—连接两个或更多的数组，并返回结果。
    every()—检测数组元素的每个元素是否都符合条件。
    some()—检测数组元素中是否有元素符合指定条件。
    filter()—检测数组元素，并返回符合条件所有元素的数组。
    indexOf()—搜索数组中的元素，并返回它所在的位置。
    join()—把数组的所有元素放入一个字符串。
    toString()—把数组转换为字符串，并返回字符串。
    lastIndexOf()—返回一个指定的字符串值最后出现的位置，在一个字符串中的指定位置从后向前搜索。
    map()—通过指定函数处理数组的每个元素，并返回处理后的数组。
    slice()—选取/截取数组的的一部分，并返回一个新数组。
    valueOf()—返回数组对象的原始值。
