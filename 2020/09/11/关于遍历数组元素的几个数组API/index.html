数组API

1.forEach遍历数组所有元素

输出： 0 1; 1 2; 2 3; 3 4; 4 5;

2.every用来判断所有的数组元素是否都符合条件，有一个元素不符合条件就返回false

输出：false

3.some用来判断所有的数组元素，只要有一个元素满足条件就会返回true

输出：true

4.sort排序，返回数组，且原数组改变

5.map对数组元素重新组装，生成新数组，但原数组不变

输出：[“1“, “2“, “5“, “4“, “3“]

6.filter通过某一个条件过滤数组，过滤符合条件的数组元素，返回新数组，但原数组不变
