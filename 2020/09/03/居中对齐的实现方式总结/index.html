HTML

优缺点

优点:不用考虑元素的宽度以及属性,可以实现任意元素的水平居中,因为使用绝对定位的元素都会变为内联块元素
缺点:存在兼容性问题

如何实现垂直居中

方法1:针对固定高度的任意元素(块/内联块/内联均可),使用绝对定位absolute和负边距

CSS

HTML

方法2:针对不固定高度的任意元素(块/内联块/内联均可),使用绝对定位absolute+transform: translateY(-50%)

CSS

HTML

对于使用绝对定位的垂直居中,当父元素没有设置高度时,相对定位的父元素要设置一个高度,或者让该元素相对于浏览器页面定位,因为子块元素不设置宽度会默认充满父元素的宽度,而没有设置高度则默认为0

方法3:针对不固定高度的任意元素(块/内联块/内联均可),flex布局 + align-items: center;设置垂直轴对齐方式

CSS

HTML

父元素使用flex布局,则子元素都会变为块元素,都可以设置宽高

优缺点

优点：flex布局灵活，不需要对子元素进行任何的设置
缺点：具有兼容性。子元素的float、clear、position等属性无法使用，如果同时具有其他布局，容易产生影响。

方法4:针对单行文本的垂直居中,使用line-height行高

HTML

方法5:针对单行/多行文本的垂直居中,使用display:table将父容器设置为块状表格,使用display:table-cell将子元素设置为表格单元格,使用vertical-align: middle;属性将表格单元格里的文本垂直居中

CSS

HTML

display:table-cell表格单元格是不单独占一行的,可以设置宽高,类似于内联块元素
