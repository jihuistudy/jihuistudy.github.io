【HTML】HTML标签嵌套规则

发表于 2021-03-22 分类于 前端
本文字数： 934 阅读时长 ≈ 1 分钟

HTML标签元素分类

  • 块级元素 <div>,<p>,<h1>~<h6>,<ul>,<ol>,<li>,<dl>,<dt>,<dd>,<br/>,<table>，可以套某些块标签和行标签

    • 块级元素的特点
      (1) 一个标签占一行
      (2)高度，宽度，外边距以及内边距都可以控制
      (3)宽度默认是容器(父级宽度)的100%
      (4)是一个容器及盒子，里面可以放行内元素和块级元素

  • 内联元素(行内元素) <a>,<span>,<label>,<em>,<strong>,<b>,<i>，只能套用行标签

    • 行内元素的特点
      (1)相邻行内元素在一行上，一行可以显示多个。
      (2)高、宽直接设置是无效的。
      (3)默认宽度就是它本身内容的宽度。
      (4)行内元素只能容纳文本或者其他行内元素。

  • 内联块元素 <input/>,<img/>,<td>

    • 和相邻行内元素（行内块）在一行上,但是之间会有空白缝隙。一行可以显示多个
    • 默认宽度就是它本身内容的宽度。
    • 高度，行高、外边距以及内边距都可以控制。

  • 空元素（单标签元素）<br>,<hr>,<img>,<input>,<link>,<meta>

HTML标签嵌套规则

  • 块级元素可包含行内元素和某些块元素。

  • 行内元素不能包含块元素，只能包含其他行内元素或文本。

    • 链接里面不能再放链接
    • 特殊情况下，<a>标签里面可以放块级元素，但是最好要将<a>标签转换成块级模式。

  • 块级元素不能放在<p>标签内。

  • 特殊块级元素只能包含行内元素，不能包含块级元素，如:h1,h2,h3,h4,h5,h6,p,dt。

    • 只有文字才能组成段落因此<p>标签里面不能放块级元素，特别是<p>标签内不能放<div>标签。
    • 同理还有这些标签h1,h2,h3,h4,h5,h6,dt，他们都是文字类块级标签，里面不能放其他块级元素。
